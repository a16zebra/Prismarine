## 🧠 What is Prismarine?

Prismarine lets every folder define its own UI with `.index.p` YAML files.  
Think of it like: *"What if Notion met your file system?"*

## 🛠 Features

- Local-first, Git-friendly
- YAML-defined Prism UI
- Plugin system and WYSIWYG editor (in progress)
- Scriptable actions from folders
